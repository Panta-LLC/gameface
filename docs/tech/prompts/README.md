## Conventions

- Each prompt includes: Context, Objectives, Acceptance Criteria, and a Test Plan.
- Keep commit messages scoped to the prompt’s focus (feat/fix/chore), reference Epic/Story if applicable.
- Update dashboards, runbooks, and docs as specified in prompts to keep operational readiness.
